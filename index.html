  1. Which CSS selector is more powerfull. Inline is more powerfull because it cascade all other style. Then internal if style set after css link.
    1. Inline: write in html respective line.
      For Example <div style="width: 100px">
    2. Internal: set on respective HTML header section.
      For Example: <style> .some-style{width:100px;} </style>
    3. External: write css code in different file and link up to targeted HTML which should link on header section.
      For Example: <link rel="stylesheet" href="style.css">
  2. HTML read the Style first which set in head, before read html code. For applying that style before presenting content. Otherwise first content will appear witout style and then apply it.
  3. Script (bevhaior) place on last, because bevabior set after loaded everything. Like first set image size and then apply slide.
  4. In css style we have to optimize code, means within minimum code we have to apply style. For example: border: 5px dashed red;
  5. There are three type Selectors which is mostly used in HTML styling as belows
    1. Element: div, body, section, aside, nav, etc. This selector is not specific, because style on element/tag selector apply on all same element/tag's present in whole html document.
    2. Class: user define class, multiple use in HTML code, more specific. Selector Class name should be start with letter like <h1 class="heading">. In css we have to write as .heading{color:red;}, mean starting with DOT(.). It used to apply same style on multiple tag/element's
    3. ID: userdefine unique id, single use in HTML code, more & more specific. Selector ID should be written as <h1 id="heading"> in HTML and in CSS as #heading{class:red;}. In CSS it always star with HASH(#).
    Note: Multiple class can be used in same line with a space between the class. Like <h1 class="style1 style2">. It used to apply multiple css style group on same object.
  6. Color code: There are several ways to define color of object as belows,
    • Solid: directly write the color name and it will appear. Example: color: red; color: green; color: yellow;
    • Hexa (mostly used): Three main color red, green, blue write as two digit hexa code. This code is widely available on internet, photoshop, etc. Example: color: #000000; Color: #FFFFFF;
    • rgb: This is a function type code, which format is rgb(00,00,00) for black.
    • rgba: This same as like rgb, but with a extra properties that is aplha. Which control the transparency of that color. Example, rgba(00,00,00,.2), this code apply black color with 20% transparency.
